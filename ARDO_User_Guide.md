## Getting Started

### Interface Overview

ARDO features a clean, professional dark-themed interface designed to maximize your creative workspace:

- **Top Toolbar**: Access File, Edit, View, Project, and Help menus
- **Left Toolbar**: Quick access to essential design tools with keyboard shortcuts
- **Central Canvas**: Interactive 3D viewport for designing and visualizing
- **Right Panel**: Collapsible design tools panel with 9 major feature categories
- **Bottom Status Bar**: Real-time information about your current tool, grid, and project status

### Your First Project

1. Click **File** → **Projects** to open the Project Manager
2. Click **"New Project"** and enter a name and description
3. Your new project opens in the 3D viewport
4. Start designing using the tools in the left toolbar or right panel

## Core Features

### 1. 3D Model Viewer & Design Canvas

The heart of ARDO is the interactive 3D viewport powered by Three.js.

**Features:**
- Real-time 3D visualization with professional lighting
- Orbit controls (drag to rotate, scroll to zoom)
- Infinite grid system for spatial reference
- Support for multiple geometric primitives

**Loading 3D Models:**
1. Click **File** → **Load 3D Model**
2. Drag and drop your GLTF, GLB, OBJ, or FBX file
3. Or click **"Browse Files"** to select from your computer
4. Try the sample models for quick testing

### 2. AI Design Partners (14 Personalities)

Chat with 14 specialized AI architects powered by OpenRouter API.

**Available Partners:**
- **Zaha Hadid**: Parametric & organic forms
- **Norman Foster**: Sustainable high-tech architecture
- **Frank Lloyd Wright**: Organic architecture & nature harmony
- **Modernist**: Clean lines & minimalism
- **Structuralist**: Engineering & structural integrity
- **Urbanist**: City-scale & public spaces
- **Traditionalist**: Classical proportions & craftsmanship
- **Futurist**: Smart cities & AI integration

**Categories:**

**Drag-and-Drop Application:**
1. Open **"Material Library"** in the right panel
2. Browse or search for a material
3. **Drag** the material card onto the 3D canvas

## Tips & Best Practices

### Performance Optimization

- **Close unused panels** to maximize viewport performance
- **Use lower quality settings** for real-time work, high quality for final renders
- **Limit the number of loaded 3D models** in a single scene
- **Clear browser cache** if experiencing slowdowns

### Design Workflow

1. **Start with AI consultation** - Discuss your concept with an AI partner
2. **Load or create base geometry** - Import models or use primitives
3. **Apply materials** - Use drag-and-drop for quick material testing
4. **Run compliance check** - Catch code violations early
5. **Optimize costs** - Use the Cost Optimizer to stay within budget
6. **Render and export** - Create final visualizations

### Collaboration Tips

- **Save frequently** - Use Ctrl+S or rely on auto-save
- **Use descriptive project names** - Makes finding projects easier
- **Export multiple formats** - Share with clients in their preferred format
- **Document AI conversations** - Copy important design advice from AI chats

## Troubleshooting

### 3D Viewport Issues

**Problem**: Viewport is black or not rendering
- **Solution**: Refresh the page (F5) or check WebGL support in your browser

**Problem**: Models load slowly
- **Solution**: Use smaller file sizes or optimize models before importing

### Material Application Issues

**Problem**: Materials not applying when dragged
- **Solution**: Ensure you're dragging onto the 3D canvas drop zone

### AI Chat Not Responding

**Problem**: AI partner not replying
- **Solution**: Check internet connection; the system uses OpenRouter API

### Export Failures

**Problem**: Export button doesn't work
- **Solution**: Select a format first, then click Export
